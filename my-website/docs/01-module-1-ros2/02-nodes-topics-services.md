
## MCQs

### Question 1
What is the primary function of a ROS 2 node?
A) To manage the entire ROS 2 network.
B) To act as an executable unit of software performing a specific task.
C) To facilitate communication between different ROS 2 systems on separate machines.
D) To visualize data from ROS 2 topics.

**Correct Answer:** B

### Question 2
In ROS 2, what mechanism is used for one-way data exchange between nodes?
A) Services
B) Actions
C) Topics
D) Parameters

**Correct Answer:** C

### Question 3
Which of the following describes the role of a publisher node in ROS 2?
A) It requests a service from a service server.
B) It sends data messages to a specific topic.
C) It receives and processes data messages from a topic.
D) It provides a service to other nodes upon request.

**Correct Answer:** B

### Question 4
What is the communication pattern followed by ROS 2 services?
A) One-way data streaming.
B) Publish-subscribe model.
C) Request-response model.
D) Broadcast communication.

**Correct Answer:** C

### Question 5
If a ROS 2 client needs to perform a computational task and receive a result, which communication method should it use?
A) Publishing to a topic.
B) Subscribing to a topic.
C) Calling a service.
D) Setting a parameter.

**Correct Answer:** C
