"# Humanoid-Robotics-Textbook

This project contains a comprehensive textbook on humanoid robotics with a RAG (Retrieval-Augmented Generation) chatbot system that allows students and researchers to interactively ask questions about the content.

## Project Structure

- `rag-backend/rag-backend-chatbot/` - Backend RAG system using Cohere and Qdrant
- `my-website/` - Docusaurus-powered frontend website with chatbot integration
- `specs/` - Specification documents
- `history/` - Historical content

## Quick Start

1. **Backend Setup**: Navigate to `rag-backend/rag-backend-chatbot/` and follow the setup guide in SETUP_GUIDE.md
2. **Frontend Setup**: Navigate to `my-website/` and follow the Docusaurus setup instructions
3. **Start Services**: Run both backend and frontend to enable the full chatbot functionality

## Features

- Interactive Q&A chatbot that understands textbook content
- Vector database for efficient information retrieval
- Context-aware responses based on selected text
- Streaming responses for real-time interaction"
