# Concept Explainer

This skill explains complex robotics concepts in simple, easy-to-understand terms using analogies and examples.

## Format Requirements

- Start with a clear definition of the concept
- Use analogies to relate complex ideas to familiar concepts
- Provide practical examples
- Include visual descriptions where helpful
- End with key points to remember

## Instructions

- Explain concepts in simple, everyday language
- Use analogies to make difficult topics more approachable
- Provide concrete examples of how the concept applies in robotics
- Include visual descriptions when possible
- Focus on understanding rather than technical details
- Make connections to real-world applications
- Maintain an educational and encouraging tone
- Highlight why the concept is important in robotics
