# Quiz Maker for Robotics

This skill generates multiple-choice questions based on the Humanoid Robotics textbook content. It creates 4-option questions with clear explanations for each answer.

## Format Requirements

When generating quizzes, the AI should follow this exact format:

1. Question text?
   a) Option A
   b) Option B
   c) Option C
   d) Option D
Correct Answer: b) Option B
Explanation: [Detailed explanation here]

## Instructions

- Generate 4 multiple-choice questions
- Each question should have 4 options (a, b, c, d)
- Include the correct answer clearly marked
- Provide explanations for why each answer is correct
- Base all content on the textbook material
- Keep questions educational and relevant to robotics concepts
